# Hospital Patient Management System

## 📌 Project Overview
This project is a SQL-based hospital database system.

## 📊 Features
- Patient Management
- Doctor Management
- Appointments Tracking
- Billing System

## 📁 Files
- schema.sql → Tables
- data.sql → Sample Data
- queries.sql → Basic Queries
- insights.sql → Analysis Queries

## 🚀 Purpose
To simulate real-world hospital database operations.
